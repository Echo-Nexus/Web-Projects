# SwiftBasket – A Full-Stack E-Commerce Platform

- Modern full-stack e-commerce application built with the MERN Stack.
- Designed to provide a seamless and responsive online shopping experience.
- Features secure user authentication, product browsing.
- Implements RESTful APIs for efficient communication between the frontend and backend.
- Uses MongoDB for storing products, users, and order data.
- Built with scalability, performance, and user experience in mind.

## 🚀 Features

- User Registration & Login
- Secure Authentication & Authorization
- Product Listing and Details
- Shopping Cart Functionality
- Product Categories
- Responsive Design
- REST API Integration
- Database Management with MongoDB

## 🛠️ Tech Stack

- Frontend: EJS, React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub
- Deployment: Render

## 📚 What I Learned

- Building REST APIs with Express.js
- Connecting Frontend and Backend
- Managing Data with MongoDB
- Authentication and Security Basics
- Deploying Full-Stack Applications
- Using Git and GitHub for Version Control

## 🎯 Project Goal

- To develop a real-world e-commerce platform.
- To strengthen full-stack web development skills.
- To gain hands-on experience with the MERN Stack.
- To understand modern web application architecture.
